# Retain retouch provenance for cross-channel guidance

Prokudin will retain per-pixel retouch provenance and confidence with its working state. Later Guided Healing may use a high-confidence repair as guide data, but excludes an uncertain repair and every clone-stamped pixel; this preserves verified recovered structure while preventing a speculative or manually copied result from propagating into another colour channel. Projects that predate this data load with Unknown Provenance: they require both guides to agree and are never a sole guide. Provenance follows a crop, resets on reimport or new alignment, restores atomically with Undo or Redo, and persists in explicit saves and autosaves. Always using original data would discard useful repairs, whereas always using the latest working data would spread earlier errors.
